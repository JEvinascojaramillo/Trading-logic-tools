# 📈 Finance & Risk Logic (JS)

A collection of JavaScript functions designed to handle professional trading calculations, focusing on **Compound Interest** and **Risk Management**.

## 🚀 Key Features
- **Compound Interest Engine:** Calculates account growth over time based on a fixed percentage.
- **Risk Validator:** Checks if the risk per trade exceeds professional standards (3%) and suggests position sizing.
- **Clean Code:** Written with modern JavaScript (ES6) syntax.

## 🛠️ How to use
1. Clone the repository.
2. Run `node finance_calculator.js` in your terminal or copy the code into your browser's console.
